# Projeto de Gerência de Configuração

Este repositório tem como objetivo demonstrar, na prática, o uso de ferramentas de controle de versão e gerência de configuração de software.  
Aqui serão armazenados os documentos, imagens e artefatos desenvolvidos pela equipe ao longo do projeto.  

O propósito é manter um histórico organizado das alterações, facilitar o trabalho colaborativo entre os integrantes e garantir o controle das versões de cada componente do sistema.
